# first-app

`A small React portfolio site using Tailwind CSS and Font Awesome. This project demonstrates a simple router-based layout, reusable components, and responsive styling.`

## Live Demo

[`first-app`](https://start-react-tawny.vercel.app/)

## Features

- Client-side routing via React Router (`createBrowserRouter`) — see [`App`](src/App.jsx).
- Main layout with header/nav and footer — see [`MainLayout`](src/layouts/main-layout.jsx).
- Reusable components:
  - Navigation: [`Nav`](src/components/nav.jsx) and [`Li`](src/components/li.jsx)
  - Title + decorative SVG: [`Title`](src/components/title.jsx) and [`LineStar`](src/components/svg.jsx)
  - Cards with modal preview: [`Card`](src/components/card.jsx)
  - Form inputs: [`Input`](src/components/input.jsx)
  - Footer: [`Footer`](src/components/footer.jsx)
  - Small paragraph component: [`P`](src/components/p.jsx)
- Pages: home, about, portfolio, contact — see [`src/pages/home.jsx`](src/pages/home.jsx), [`src/pages/about.jsx`](src/pages/about.jsx), [`src/pages/portfolio.jsx`](src/pages/portfolio.jsx), [`src/pages/contact.jsx`](src/pages/contact.jsx).
- Tailwind CSS for utility-first styling — Tailwind is imported in [`src/index.css`](src/index.css).
- Static assets served from `public/` (images) and referenced in components (e.g. `/images/avataaars.svg`, `/images/port1.png`...).

## How it works (high level)

- `src/main.jsx` mounts the React app.
- `src/App.jsx` defines the route tree using React Router and mounts the `MainLayout` which renders an `<Outlet />` for page content.
- `Nav` toggles a mobile menu using refs and CSS utility classes. Links are generated with [`Li`](src/components/li.jsx) using `NavLink`.
- `Card` displays thumbnails and opens a full-screen preview overlay on click (managed via a `ref`).
- `Title` composes a heading with the decorative star line generated by `LineStar` (`svg.jsx`), which programmatically generates a star path.
- Styling and utilities come from Tailwind; Font Awesome icons are imported in the nav component.
